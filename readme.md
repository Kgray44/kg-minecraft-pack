# KG Minecraft Modpack

A large modpack focused on automation, exploration, military, ships, trains, space travel, engineering, and much more!

Mods are automatically downloaded and kept up to date using **packwiz**.

---

# Installation (Prism Launcher)

### 1. Install Java 21

Download and install Java Version 21 (DO NOT install Java 17 or Java 25):

[Download Java 21](https://www.oracle.com/java/technologies/downloads/#java21)

---

### 2. Install Prism Launcher

Download and install Prism Launcher:

[Download Prism Launcher](https://prismlauncher.org/download/)

---

### 3. Download the Modpack

Download the latest released Prism instance file found in *Releases*:

[**KG_Modpack_Prism_Instance.zip**](https://github.com/Kgray44/kg-minecraft-pack/releases)


---

### 4. Import the Instance

Open **Prism Launcher**

Click:

*Add Instance*

Then choose:

*Import*

Select from downloads:

*KG_Modpack_Prism_Instance.zip*

Prism will create the instance automatically.

---

### 5. Launch the Pack

Click **Play**.

The first launch will automatically:

- download all mods

- install dependencies

- configure the pack

This may take a few minutes.

---

# Optional Graphics Mods (IMPORTANT)

Some graphics mods can be heavy on weaker computers.

If your game crashes or runs poorly:

Right click the instance in Prism:

*Edit*

Then click (in the left menu):

*Mods*

You can disable heavy graphics mods like:

- Distant Horizons

- Iris Shader Support

Disabling these will improve performance.

---

# Updating the Pack

Updates are automatic.

Every time the game launches:

- packwiz checks for updates

- new mods download automatically

No manual updates required.

---

# Recommended Settings

For best performance:

- RAM: 5 GB minimum

- RAM: 8-10 GB recommended

- Java: Version 21

*(Will not run with 8GB computer ram total)

These can be adjusted in:

*Prism Launcher → Edit Instance → Settings*

---

# Troubleshooting

### Game crashes on startup

Disable heavy graphics mods:

*Edit Instance → Mods*

Turn off:

- Distant Horizons
- Iris Shaders

---

### Mods won't download

Check your internet connection and relaunch the game.

The installer will retry automatically.
